# Product: Conductor Extension Demo

## Vision

To provide a clear, functional starter project that demonstrates how to use the
Conductor extension for the Gemini CLI to facilitate rapid application
prototyping. This repository serves as a hands-on guide for developers looking
to accelerate their workflow from idea to initial code.

## Target Audience

- Developers interested in learning and adopting AI-assisted development
  workflows.
- Users of the Gemini CLI who want to understand the practical application of
  the Conductor extension.
- Teams looking for a structured yet rapid process for validating application
  ideas.

## Core Demonstration

This project showcases an end-to-end, AI-driven prototyping workflow facilitated
by the Conductor extension:

1.  **Idea to Specification:** Using `/conductor:newTrack` to define a new
    feature or idea, which generates a detailed specification.
1.  **Specification to Code:** Using `/conductor:implement` to automatically
    generate a working Python Flask application or implement a feature based on
    the specification from the new track.
1.  **Iterative Development:** Using the streamlined `workflow.md` as a guide
    for making subsequent changes and additions to the generated prototype.
1.  **Conductor-Managed Project:** Demonstrating how the Conductor extension
    helps maintain project context through documents like `product.md`,
    `tech-stack.md`, and `workflow.md`.

## Key Takeaways for Users

- A practical understanding of how to structure a project for the Conductor
  extension.
- A working example of how to utilize Conductor's built-in commands like
  `/conductor:newTrack` and `/conductor:implement` for workflow automation.
- A template for a rapid, prototype-focused development process that can be
  adapted for other projects.

## Success Metrics

- **Clarity:** A new user can easily clone the repository, read the
  documentation, and understand the workflow.
- **Functionality:** The user can successfully execute the custom commands and
  run the generated prototype.
- **Adaptability:** The user feels confident in using this project's structure
  as a template for their own application ideas.
